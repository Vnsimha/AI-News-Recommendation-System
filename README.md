**AI News Recommendation** 

**Overview**

This project is a Personalized AI News Recommendation System that suggests news based on user interests and provides AI-generated summaries.

**Features**

* User Authentication (Login/Register)
* Personalized News Feed
* NewsAPI Integration
* AI-based Text Summarization
* Dashboard Analytics

**Tech Stack**

* Python
* Gradio
* Transformers (Hugging Face)
* NewsAPI
* Bcrypt
